# 虚拟仿真实训平台 - 前端页面

基于 Vue 3 + Vite 的前端应用，为学生、教师、管理员三种角色提供实训管理和 AI 辅助功能。

## 技术栈

- **框架**: Vue 3（组合式 API + TypeScript）
- **构建**: Vite
- **UI 组件**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router（Hash 模式）
- **HTTP**: Axios（请求拦截器自动注入 Token）
- **包管理**: pnpm
- **运行环境**: Node.js ≥ 24

## 快速启动

```bash
# 1. 安装依赖
pnpm install

# 2. 启动开发服务器
pnpm run dev
```

前端运行在 `http://localhost:5173`，默认请求 `http://localhost:3000/api` 的后端服务。

## 环境变量

| 文件 | 生效场景 | 说明 |
|------|----------|------|
| `.env.development` | `pnpm run dev` | 开发环境，baseURL 指向本地后端 |
| `.env.production` | `pnpm run build` | 生产环境，baseURL 指向生产后端域名 |

两个文件中的 `VITE_API_BASE_URL` 决定了前端请求后端 API 的地址。只有 `VITE_` 前缀的变量才会暴露给前端代码。

## 目录结构

```
src/
├── api/                后端接口请求函数（按模块分文件）
├── views/              页面组件
│   ├── admin/          管理员页面（用户管理）
│   ├── teacher/        教师页面
│   ├── student/        学生页面（实训任务 + AI 助手）
│   └── training/       实训管理页面（项目管理、任务分配、提交查看）
├── components/         公共组件（布局、侧边栏）
├── router/             路由配置 + 导航守卫
├── stores/             全局状态（用户信息、Token）
└── config/             菜单配置（按角色动态渲染）
```

## 角色与功能

| 角色 | 可见功能 |
|------|----------|
| **学生** | 我的实训任务、提交报告、AI 助手（智能问答） |
| **教师** | 实训项目管理、任务分配、查看提交、AI 批阅、人工复核 |
| **管理员** | 用户管理 + 教师全部功能 |

## 打包部署

```bash
pnpm run build     # 输出到 dist/ 目录
pnpm run preview   # 本地预览打包结果
```
