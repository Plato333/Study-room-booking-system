# 自习室预约系统 — 实验报告

**课题名称：基于 NestJS + Vue 3 的自习室预约系统设计与实现**

---

## 1. 前言

### 1.1 课题背景与意义

随着高校招生规模的不断扩大，图书馆和自习室的座位资源日益紧张。"占座"现象屡禁不止，学生常常需要亲自到现场才能知道是否有空位，浪费了大量时间。传统的自习室管理方式存在以下痛点：

1. **信息不透明**——学生无法远程查看座位使用情况，只能到现场碰运气
2. **管理效率低**——管理员缺乏有效的工具来管理座位资源
3. **资源浪费**——占座导致的座位利用率低下

因此，开发一套自习室预约系统具有重要的现实意义。本系统旨在通过信息化手段，实现座位的线上预约、签到和释放，提高自习室资源利用率，为学生提供便捷的学习环境。

### 1.2 本课题主要工作

本课题完成了一个全栈自习室预约系统的设计与开发，主要工作包括：

1. **需求分析**：分析自习室预约的业务流程，明确管理员和学生两类角色的功能需求
2. **系统设计**：完成系统架构设计、数据库设计、API 接口设计、前端路由设计
3. **后端开发**：基于 NestJS + TypeORM + MySQL 实现 RESTful API，包括用户认证（JWT）、自习室 CRUD、座位 CRUD、预约管理、签到释放、数据统计、AI 助手等模块
4. **前端开发**：基于 Vue 3 + Vite + Element Plus 实现用户界面，包括管理端看板、学生端预约流程、全局毛玻璃 UI 风格
5. **AI 接入**：接入 DeepSeek Chat 大模型，实现自然语言问答助手
6. **测试验证**：通过 Apifox 进行接口测试，验证各模块功能正常

## 2. 需求分析

### 2.1 需求分析概述

自习室预约系统的目标是解决高校自习室座位资源管理问题。系统面向两类用户：**管理员**和**学生**。管理员负责管理自习室和座位资源，查看预约数据；学生可以通过系统查找空闲座位、预约、签到和释放座位。

系统的核心业务流程为：

```
学生登录 → 选择日期和时段 → 查看空闲座位列表 → 预约座位
    ↓
按时到达自习室 → 扫码签到 → 座位状态变为占用
    ↓
学习结束 → 手动释放座位 → 座位恢复空闲
```

## 4. 系统详细设计

### 4.1 用户认证模块设计

**模块功能：** 实现用户注册、登录认证、Token 验证、角色权限控制。

**流程图：**

```
用户提交登录表单
    ↓
AuthController.login()
    ↓
AuthGuard('local')  →  LocalStrategy.validate()
    ├─ 根据用户名查找用户
    ├─ bcrypt.compare() 比对密码
    └─ 返回 { id, username, role }
    ↓
AuthService.login()
    ├─ jwtService.sign({ sub: id, username, role })
    └─ 返回 { access_token, user }
    ↓
后续每次请求：
Authorization: Bearer <token>
    ↓
AuthGuard('jwt')  →  JwtStrategy.validate()
    ├─ 解析 Token
    └─ 返回 user 对象注入 req.user
    ↓
RolesGuard  →  检查 @Roles() 元数据 → 比对 req.user.role
```

**关键代码：**

```typescript
// JWT 策略：解析 Token
@Injectable()
export class JwtStrategy extends PassportStrategy(Strategy) {
  constructor(configService: ConfigService) {
    super({
      jwtFromRequest: ExtractJwt.fromAuthHeaderAsBearerToken(),
      secretOrKey: configService.get('JWT_SECRET'),
    });
  }

  async validate(payload: { sub: number; username: string; role: string }) {
    return { id: payload.sub, username: payload.username, role: payload.role };
  }
}
```

```typescript
// 角色守卫：权限校验
@Injectable()
export class RolesGuard implements CanActivate {
  constructor(private reflector: Reflector) {}

  canActivate(context: ExecutionContext): boolean {
    const roles = this.reflector.getAllAndOverride<string[]>(ROLES_KEY, [
      context.getHandler(),
      context.getClass(),
    ]);
    if (!roles) return true;
    const { user } = context.switchToHttp().getRequest();
    return roles.includes(user?.role);
  }
}
```

### 4.2 预约管理模块设计

**模块功能：** 实现空闲座位查询、预约创建（含冲突检测）、取消预约、签到、释放。

**预约流程图：**

```
学生选择日期 + 开始时间 + 结束时间
    ↓
系统查询空闲座位（排除已冲突预约）
    ↓
展示空闲座位列表
    ↓
学生选择座位 → 点击预约
    ↓
冲突检测：
├─ 该座位是否存在？ → 否 → 提示"座位不存在"
├─ 座位状态是否空闲？ → 否 → 提示"不可预约"
├─ 结束时间 > 开始时间？ → 否 → 提示"时间不合法"
├─ 同一座位同一日期是否有未取消的预约
│  且时间段重叠？ → 是 → 提示"该时段已被预约"
└─ 全部通过 → 创建预约记录
    ↓
预约成功
```

**冲突检测核心代码：**

```typescript
async create(dto: CreateReservationDto, studentId: number) {
  // 校验座位存在且空闲
  const seat = await this.seatRepository.findOne({ where: { id: dto.seat_id } });
  if (!seat) throw new NotFoundException('座位不存在');
  if (seat.status !== SeatStatus.AVAILABLE)
    throw new BadRequestException('该座位当前不可预约');
  if (dto.end_time <= dto.start_time)
    throw new BadRequestException('结束时间必须晚于开始时间');

  // 冲突检测：同一座位、同一天、时间段重叠
  const conflict = await this.reservationRepository.findOne({
    where: {
      seat_id: dto.seat_id,
      date: dto.date,
      status: Not(ReservationStatus.CANCELLED),
      start_time: LessThanOrEqual(dto.end_time),
      end_time: MoreThanOrEqual(dto.start_time),
    },
  });
  if (conflict) throw new BadRequestException('该时段已被预约');
  // ...创建预约
}
```

## 5. 系统实现

### 5.1 用户认证功能

**实现思路：** 使用 NestJS Passport 模块，通过 LocalStrategy 验证用户名密码，JwtStrategy 验证 Token。用户注册时使用 bcrypt 对密码进行哈希加密。

**注册功能关键代码：**

```typescript
// users.service.ts — 注册
async register(dto: CreateUserDto) {
  // 检查用户名唯一性
  const existing = await this.userRepository.findOne({
    where: { username: dto.username },
  });
  if (existing) throw new ConflictException('用户名已存在');

  // bcrypt 加密密码
  const hashedPassword = await bcrypt.hash(dto.password, 10);
  const user = this.userRepository.create({
    ...dto,
    password: hashedPassword,
  });
  const saved = await this.userRepository.save(user);
  const { password, ...result } = saved; // 排除密码字段
  return result;
}
```

### 5.2 座位预约功能

**实现思路：** 预约创建时进行多维度的校验（座位存在性、状态、时间有效性、冲突检测）。空闲座位查询使用 SQL 子查询排除已被预约的座位。

**空闲座位查询关键代码：**

```typescript
// seats.service.ts — 查询空闲座位
async findAvailable(date: string, startTime: string, endTime: string, roomId?: number) {
  const qb = this.seatRepository
    .createQueryBuilder('seat')
    .innerJoinAndSelect('seat.room', 'room')
    .where('room.status = :roomStatus', { roomStatus: 'open' })
    .andWhere('seat.status = :seatStatus', { seatStatus: SeatStatus.AVAILABLE });

  if (roomId) {
    qb.andWhere('seat.room_id = :roomId', { roomId });
  }

  // 排除已有冲突预约的座位
  qb.andWhere(
    `seat.id NOT IN (
      SELECT r.seat_id FROM reservations r
      WHERE r.date = :date
        AND r.status != :cancelledStatus
        AND r.start_time < :endTime
        AND r.end_time > :startTime
    )`,
    { date, cancelledStatus: ReservationStatus.CANCELLED, endTime, startTime },
  );

  return qb.orderBy('room.id', 'ASC').addOrderBy('seat.id', 'ASC').getMany();
}
```

### 5.3 AI 智能助手功能

**实现思路：** 接收用户自然语言提问，收集系统实时数据作为上下文，调用 DeepSeek API 生成回答。系统提示词（System Prompt）包含当前座位数、预约数、自习室信息等，让 AI 能准确回答业务相关问题。

**AI 聊天关键代码：**

```typescript
// ai.service.ts — 调用 DeepSeek API
async chat(message: string) {
  const context = await this.buildContext(); // 收集实时数据
  const reply = await this.callDeepSeek(message, context);
  return { reply };
}

private async callDeepSeek(message: string, context: string): Promise<string> {
  const apiKey = this.configService.get<string>('AI_API_KEY');
  const baseUrl = this.configService.get<string>('AI_BASE_URL');

  const res = await fetch(`${baseUrl}/v1/chat/completions`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${apiKey}`,
    },
    body: JSON.stringify({
      model: 'deepseek-chat',
      messages: [
        {
          role: 'system',
          content: `你是自习室预约系统的 AI 助手。当前系统状态：\n${context}`,
        },
        { role: 'user', content: message },
      ],
      temperature: 0.7,
      max_tokens: 800,
    }),
  });

  const data = await res.json();
  return data.choices?.[0]?.message?.content;
}
```
